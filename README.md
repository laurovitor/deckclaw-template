# deckclaw-template

Template visual estático para referência de UX/UI do novo DeckClaw.

## Objetivo
Alinhar linguagem visual, componentes e padrões de navegação antes da implementação funcional.

## Páginas
- index.html
- login.html
- register.html
- terms.html
- widgets.html
- forms.html
- tables.html
- kanban.html
- users.html
- openclaw.html
- tasks.html
- reports.html
- settings.html

## Design system
- `assets/styles/tokens.css`
- `assets/styles/base.css`
- `assets/styles/components.css`

## Como abrir
Abra `index.html` no navegador (ou use servidor local simples).

## Checklist mínimo coberto
- Sidebar colapsável
- Botões (normal/loading/disabled)
- Inputs com focus/error
- OTP slots
- Tabelas/cards/modais/toasts
- Responsividade para mobile/tablet/desktop
